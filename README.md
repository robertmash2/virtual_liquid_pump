# Liquid Conduits - Instant Liquid Transport

Tired of hauling water buckets across your world? Liquid Conduits adds a simple solution for transporting liquids instantly across any distance.

## Features

- Craft paired conduit blocks that link together automatically
- Place one conduit at your water source, another wherever you need water
- Pour water into the intake conduit - it instantly appears at the output
- Works across any distance, even between dimensions
- Powered by coal for balanced gameplay
- Clean design with no visible pipes cluttering your builds

## How it Works

Think of it like an ender chest, but for liquids. Conduit pairs are crafted together and automatically linked. Right-click the intake conduit to add coal fuel, then use a water bucket on it to instantly transfer water to the paired output location.

## Perfect for

- Remote farm irrigation
- Base water supply systems
- Nether ice farms
- Large building projects
- Any situation where hauling water buckets is tedious

## Requirements

- Minecraft Java Edition 1.21+
- Fabric Loader 0.17.3+
- Fabric API

## Installation

1. Install [Fabric Loader](https://fabricmc.net/use/)
2. Download [Fabric API](https://modrinth.com/mod/fabric-api)
3. Download Liquid Conduits from [Releases](https://github.com/robertmash2/virtual_liquid_pump/releases)
4. Place both mods in your `.minecraft/mods` folder
5. Launch Minecraft with the Fabric profile

## Development

This mod is built with:
- Minecraft 1.21.10
- Fabric Loader 0.17.3
- Fabric API 0.136.0+1.21.10
- Java 21

### Building from Source

```bash
git clone https://github.com/robertmash2/virtual_liquid_pump.git
cd virtual_liquid_pump
./gradlew build
```

The compiled mod JAR will be in `build/libs/`
